# Conversor de Markdown para HTML

Converter um ficheiro de Markdown para HTML, nomeadamente:

- Cabeçalhos
- Listas ordenadas
- Negrito
- Itálico
- Links
- Imagens

Também foram implementados linhas horizontais e listas desordenadas.
